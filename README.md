# Rowly
**Rowly** - веб-приложение для игры в судоку с акцентом на визуал, прогресс и обучение. Слоган продукта: *Master the Grid*.
**Продакшен:** [https://rowly.vercel.app/](https://rowly.vercel.app/)
---
## О продукте
Rowly объединяет классическую логику судоку с современным интерфейсом (светлая и тёмная темы, адаптация под мобильные устройства), личным кабинетом и социальными механиками.

### Возможности
- **Игра** - несколько уровней сложности, таймер, жизни, заметки в ячейках, отмена хода, подсказки, встроенный «тренер» на базе Google Gemini (краткие советы по позиции).
- **Practice Mode**  отдельный режим с **чатом ИИ-тренера**: разбор доски, объяснение стратегий, подсказки по выбранной клетке (с учётом эталонного решения для точности объяснений).
- **Ежедневный челлендж** и быстрый старт с дашборда.
- **Магазин** - скины доски и титулы за внутриигровую валюту (монеты за победы).
- **Статистика и достижения** - серии дней, лучшие времена, история партий.
- **Таблица лидеров** - синхронизация с Supabase.
- **Аккаунт** - регистрация и вход через Supabase Auth (email, Google); **облачный профиль** (`profiles` + `stats_json`): ник, статистика и настройки подтягиваются после входа на другом устройстве (см. SQL в репозитории).

### Переменные окружения
В `sudoku-app/.env` (не коммитить секреты):
- `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY` - бэкенд Auth и БД.
- `VITE_GEMINI_API_KEY` - подсказки и чат в Practice Mode.
Схема таблиц Supabase: [`sudoku-app/supabase-schema.sql`](./sudoku-app/supabase-schema.sql).
### Письмо подтверждения регистрации
HTML-шаблон для письма (в т.ч. Supabase *Confirm signup*): [`sudoku-app/email-templates/confirm-registration.html`](./sudoku-app/email-templates/confirm-registration.html).
